Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML stands for Hypertext Markup Language.
  • Metadata is the data contained in the header, this information is not visible to a user of a webpage.
  • !DOCTYPE declarations are not cas-sensiitive, but HTML elements are.
  • Child elements are elements that are nested inside other elements. ie Body elements.
  • Elements refer to the entire element from opening tag to closeing tag.
  • Tag refers to only what appears inside the angle brackets.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A link element creates a connection to the external file.
  • The rel attribute specifies the relationship between the current document and the linked document.
  • The href attribute specifies the location (ie URL) of the external resource.
  • The 3 ways to style a webpage using CSS are: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • inline CSS allows you to style an element directly in the HTML file.
  • internal CSS contains the CSS rules for the webpage in the head section of the HTML file.
  • external CSS is the most common way to style a webpage
  • DRY or (do not repeat yourself) is a coding principle that encourages developers to reuse and/or share peices of code in order to reduce the number of lines of code that are written or maintained.

  • Git is a version control system that lets you manage and track ceach version of your project.
