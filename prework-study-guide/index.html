HTML

  • Metadata is the data contained in the header, this information is not visible to a user of a webpage.
  • !DOCTYPE declarations are not cas-sensiitive, but HTML elements are.
  • Child elements are elements that are nested inside other elements. ie Body elements.
  • Elements refer to the entire element from opening tag to closeing tag.
  • Tag refers to only what appears inside the angle brackets.
